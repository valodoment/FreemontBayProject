## About the Mod
The Freemont Bay Project, based on Johan The Channel's idea for a lore-friendly NYC, includes tens of police, fire, and taxi liveries unmistakably based on New York's IRL Counterparts.

Each skin and config is based on an IRL counterpart of the applied vehicle.
